# Legal Document Search Portal

A prototype searchable legal document portal built with Streamlit, Supabase, PostgreSQL full-text search, OCR markdown text, and ArcGIS Experience Builder links.

## Features

- Full-text OCR search across legal documents
- Page-level search results with highlighted snippets
- Metadata filters for city, document type, visibility, and review status
- Linked PDF access
- Linked GIS map access through ArcGIS Experience Builder
- Supabase backend
- Search logging for usage analytics

## Tech Stack

- Python
- Streamlit
- Supabase
- PostgreSQL full-text search
- ArcGIS Experience Builder
- GIS Integration
- OCR markdown text
